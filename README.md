# ✨ Floofy Pengu - FFXIV Character Profile

Welcome to my little corner of Eorzea.

## 🌐 Live Site

[floofypengu.pages.dev](https://floofypengu.pages.dev/)

## 🧩 What Is In This Site?

- Character intro + socials
- Job levels and role progress
- Raid/content clear tracker
- Free Company section
- GPose gallery
- Optional background music

## 🛠️ Built With

- HTML
- CSS
- JavaScript

## 📁 Project Files

- index.html - page layout and sections
- style.css - desktop styling
- mobile.css - mobile styling
- script.js - all profile data plus site behavior
- images/ - artwork and visual assets
- gpose/ - GPose screenshots
- music/ - background tracks

## ✏️ Quick Content Edits

Most updates happen in script.js inside SITE_DATA:

- profile - name, title, intro, socials
- jobs - class/job levels
- content - progression cards
- fc - Free Company info
- gpose - screenshot entries
- music - track source and volume defaults

## 🚀 Deploy Flow

This repo is connected to Cloudflare Pages.

1. Edit files
2. Commit and push to main
3. Cloudflare auto-deploys
